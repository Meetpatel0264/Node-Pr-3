# Aero Page

Aero Page is a multi-page website developed using Node.js, Express.js, and EJS. The project demonstrates how to create reusable layouts, organize routes, and render dynamic pages using server-side templating.

The website includes multiple sections such as Home, About, Services, Features, Pricing, Portfolio, Blog, Testimonials, FAQ, and Contact.

---

## Technologies Used

- Node.js
- Express.js
- EJS
- HTML5
- CSS3

## Installation

### Clone Repository

```bash
git clone https://github.com/Meetpatel0264/Node-Pr-3
```

### Move to Project Folder

```bash
cd Aero-Page
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
node app.js
```

or

```bash
npm start
```

### Open Browser

```bash
http://localhost:3000
```
